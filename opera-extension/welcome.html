üìù What is RML?

RML (RazzMarkupLanguage) is like HTML but super easy! Instead of confusing brackets, use simple curly braces:

{title}My First Page{.title} {text}Hello world!{.text} {bold}Easy, right?{.bold}

üé® What is RSS?

RSS (Razz Styling Style) is like CSS but simple! Style your pages with easy commands:

[color: blue;] [size: large;] [background: yellow;]

üéì Perfect for Learning

  • Easy to read and write - no confusing symbols
  • Beginner-friendly tag names like {text} instead of <p>
  • Clear opening and closing with dots: {title}...{.title}
  • Special educational tags for lessons and exercises
  • Instant preview in Chrome with our extension
  • Converts to real HTML automatically

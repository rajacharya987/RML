üìö What is RML?

RML (RazzMarkupLanguage) is an educational markup language designed to make web development accessible for students and beginners. Instead of using complex HTML syntax with angle brackets, RML uses simple curly braces that are easier to read and understand.

üéØ Why RML?

  • Beginner-friendly: Simple {tag} syntax instead of <tag>
  • Educational focus: Built-in tags for lessons, questions, and exercises
  • Visual styling: RSS (Razz Styling Style) with [property: value;] syntax
  • Cross-browser: Works on Chrome, Firefox, Safari, Edge, and mobile
  • Real-time: See changes instantly as you type

‚úÖ Perfect for Students!

RML eliminates the complexity of HTML while teaching the same concepts. Students can focus on learning web structure and design principles without getting lost in syntax.

üìù RML Basics

Basic Syntax

RML uses curly braces for opening tags and dot notation for closing tags:

{title}My First Web Page{.title} {text}This is a paragraph of text.{.text} {bold}This text is bold{.bold}

My First Web Page

This is a paragraph of text.

This text is bold

Common Tags

RML Tag HTML Equivalent Description
{title}...{.title} <h1>...</h1> Main heading
{subtitle}...{.subtitle} <h2>...</h2> Subheading
{text}...{.text} <p>...</p> Paragraph
{bold}...{.bold} <strong>...</strong> Bold text
{italic}...{.italic} <em>...</em> Italic text

üí° Quick Tip

You can also use markdown-style shortcuts: **bold**, *italic*, __underline__, and `code`

üé® RSS Styling

RSS (Razz Styling Style) is RML's answer to CSS. Use simple bracket notation to add styles:

{title}[color: blue;][size: huge;]Styled Title{.title} {text}[background: yellow;][color: black;]Highlighted text{.text} {text}[color: red;][size: small;]Small red text{.text}

üéì Educational Tags

RML includes special tags designed for educational content:

Lesson Structure

{lesson} {title}Introduction to Web Development{.title} {question}What is HTML?{.question} {answer}HTML is the standard markup language for web pages.{.answer} {exercise} Create a simple web page with a title and paragraph. {.exercise} {tip}Remember to close all your tags!{.tip} {.lesson}

Message Boxes

{success}Great job! You completed the exercise.{.success} {warning}Be careful with this syntax.{.warning} {error}Something went wrong. Check your code.{.error} {info}This is additional information.{.info} {tip}Here's a helpful tip for you.{.tip}
Great job! You completed the exercise.
Be careful with this syntax.
Here's a helpful tip for you.

üöÄ Advanced Features

Nested Tags

You can nest tags inside each other:

{container} {header} {title}My Website{.title} {subtitle}Welcome to my site{.subtitle} {.header} {section} {text}This is the main content area.{.text} {.section} {.container}

Lists

{list} {item}First item{.item} {item}Second item{.item} {item}Third item{.item} {.list}

Links and Images

{link href="https://example.com"}Visit our website{.link} {image src="photo.jpg" alt="A beautiful photo"}
